# Data Structures in C

This repository contains C programs for various data structures and algorithms.

## Contents

### 1. Student Details & Linked List Basics
- `1a_student_details_cmdargs.c` - Student details using command line arguments
- `1b_singly_linkedlist_node.c` - Create and display single linked list node

### 2. Sorting Algorithms & Linked Lists
- `2a_quick_sort.c` - Quick Sort implementation
- `2b_doubly_linkedlist_node.c` - Create and display double linked list node

### 3. Merge Sort & Circular Linked List
- `3a_merge_sort.c` - Merge Sort implementation
- `3b_circular_linkedlist_node.c` - Create and display circular linked list node

### 4. Radix Sort & Linked List Operations
- `4a_radix_sort.c` - Radix Sort implementation
- `4b_linkedlist_search.c` - Single linked list search operation

### 5. Stack & Linked List Operations
- `5a_stack_linkedlist.c` - Stack using linked list
- `5b_linkedlist_replace.c` - Single linked list replace operation

### 6. Queue & Linked List Operations
- `6a_queue_arrays.c` - Queue using arrays
- `6b_linkedlist_deletion.c` - Single linked list deletion operation

### 7. Circular Linked List & Quick Sort
- `7a_circular_linkedlist.c` - Circular linked list creation and display
- `7b_quick_sort.c` - Quick Sort (alternative implementation)

### 8. Circular Queue & Linked List Operations
- `8a_circular_queue.c` - Circular Queue implementation
- `8b_linkedlist_insertion.c` - Single linked list insertion operation

### 9. Double Linked List & Stack
- `9a_doubly_linkedlist.c` - Double linked list creation and display
- `9b_stack.c` - Stack implementation using arrays

### 10. Complete Linked List
- `10_linkedlist_all_operations.c` - Single linked list with all operations

## How to Compile and Run

```bash
gcc filename.c -o filename
./filename
```
